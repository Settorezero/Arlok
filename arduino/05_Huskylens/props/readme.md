## Props I used for the Huskylens demo

The file _faces.pdf_ has to be printed in color on thick paper. Then you can print the _HeadsHuskylens - head_ the _HeadsHuskylens - hook_ (both stl and step files are provided), if you have a Bambulab 3D-printer you can use the _HeadsHuskylens.3mf_ file that contains all parts for making 2 puppets.  
Parts are made for fitting the 'Makey' Maker Faire robot action figure you can [download from Thingiverse](https://www.thingiverse.com/thing:331035).  
Then you must assemble all parts [this reel on Instagram](https://www.instagram.com/reel/DY6lZ8hKbpZ/) shows how to do that.
